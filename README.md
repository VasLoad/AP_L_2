### Регулярное выражение для поиска гиперссылок

```regex
<a\s+(?:[^>]*?\s)?\bhref\s*=\s*(?P<border>[\"'])(?P<url>[^\"']*)(?P=border)[^>]*?>
````

**Структура:**
* `<a\s+` - ищет начало тега `<a>` с одним или несколькими пробелами.
* `(?:[^>]?\s)?` - неименованная группа для любых символов перед `href`.
* `\bhref\s=` - ищет атрибут `href=` с обязательным пробелом перед ним.
* `(?P<quote>["'])` - именованная группа `quote`, запоминает тип кавычек (`"` или `'`).
* `(?P<url>[^"']+)` - именованная группа `url`, извлекает ссылку.
* `(?P=quote)` - проверяет, что закрывающая кавычка совпадает с открывающей через именованную группу `quote`.
* `[^>]*>` - ищет конец тега `<a>`.
